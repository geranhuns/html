Comentarios

Puedes usar comentarios para explicar tu código colocando los caracteres mostrados en la imagen.
Atajo rápido: Ctrl+K Ctrl+C

DOCTYPE

Todos los documentos deben empezar con esta declaración. Es información para que el buscador entienda qué tipo de documento espera leer.
Atajo Rápido: Html:5

a

Esta etiqueta define un hipervínculo, el cual sirve para moverse de una página a otra.

abbr

Esta etiqueta define una abreviación o acrónimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

acronym

Un acrónimo o abreviación debe ser marcada dentro de esta etiqueta.

adress

Esta etiqueta define la infotmación del dueño del documento o del artículo.

applet

This is not supported in HTML5

area

Esta etiqueta define un área dentro de una imagen, la cual puede ser seleccionable con el cursor.

article

Esta etiqueta especifica un contenido independiente agrupado por sí mismo. Un artículo debe tener sentido por sí mismo y debe ser posible su distribución independiente al resto del sitio.

aside

Esta etiqueta define un contenido al lado de un segundo donde es situado. Usualmente es para poner una barra lateral en un documento.

audio

Esta etiqueta se usa para insertar un sonido en un documento, como música u otro audio de distribución stream. El texto escrito entre la etiqueta se mostrará en los navegadores que no cuenten con un soporte de reproducción de audios.
